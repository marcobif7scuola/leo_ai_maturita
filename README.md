# 🤖 Leo.AI – Tutor Digitale per la Maturità

## 📌 Descrizione
Leo.AI è un assistente digitale sviluppato per supportare gli studenti nella preparazione all’esame di maturità.  
Il sistema utilizza modelli di intelligenza artificiale per fornire spiegazioni approfondite, riassunti e analisi di contenuti in diverse discipline.

---

## 🎯 Obiettivi del progetto
- Supportare lo studio in modo interattivo
- Fornire spiegazioni precise e contestualizzate
- Integrare fonti enciclopediche come Wikipedia
- Migliorare l’organizzazione dello studio per materie

---

## ⚙️ Tecnologie utilizzate
- **Python**
- **Streamlit** (interfaccia web)
- **API Groq (LLM)** per la generazione delle risposte
- **Wikipedia API** per contenuti informativi
- **PyPDF2** per lettura PDF
- **ReportLab** per generazione PDF

---

## 🧠 Funzionalità principali
- Chat intelligente suddivisa per materie:
  - Italiano
  - Storia
  - Matematica
  - Chimica
- Modalità di risposta:
  - Normale con integrazione Wikipedia
  - Riassunto
- Upload di file:
  - PDF
  - Immagini
- Generazione PDF delle conversazioni
- Interfaccia intuitiva e interattiva

---

## 🏗️ Architettura del sistema
Il sistema è basato su architettura client-server:
- Frontend sviluppato con Streamlit
- Backend logico gestito in Python
- Comunicazione con API tramite richieste HTTP

---

## ☁️ Deploy
L’applicazione è distribuita tramite Streamlit Community Cloud ed è accessibile via browser senza installazioni locali.
